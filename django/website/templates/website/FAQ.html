Frequently Asked Questions

Is the HSSI only for Python software?

No, the HSSI is a search interface for all software in Heliophysics and is not limited to software written in a single programming language.

My software is not included in this system. How do I add it?

Please navigate to the 'Submit a Resource' tab and add your software.

Can I submit my software if it is not hosted on GitHub?

Yes, you can use gitlab, bitbucket, or any other remote repository host.

Can I submit my software if it is not open?

Yes, but for the repository you must submit a link that takes users to a location where they can request code access.

Someone else submitted my software but there are corrections needed. How do I request edit access to the record?

Please contact the HSSI team at hssi-team@googlegroups.com to request edit access.

Can I edit my entry later if I have updated information or could not remember a detail upon initial entry?

Yes. When you submit a new resource, an email is automatically sent out to the submitter email provided with an edit link. This link will expire after a few months (date indicated in the email). If you need to edit after that time period, please reach out to hssi-team@googlegroups.com

How do I create a DOI for my software if one does not exist?

If your software is on GitHub, you can consider using the GitHub-Zenodo workflow. See this link or this link for more details. Else, you can always download all files and upload to Zenodo manually.

I have questions regarding submitting my package on the Resource Submission page. Who do I contact for help?

Please contact the HSSI team at hssi-team@googlegroups.com and provide the questions with context.

I would like to give feedback on the HSSI vocab lists / search terms. Where do I go for this?

Please navigate to the HSSI Vocab GitHub repository and submit an issue with your feedback / suggestion.
